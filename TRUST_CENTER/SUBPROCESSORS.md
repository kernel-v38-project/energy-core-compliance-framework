# Subprocessors Overview

## Definition

Subprocessors refer to external services or infrastructure components used to operate parts of the Energy_Core Enterprise platform.

## Categories

- Cloud infrastructure providers (deployment-dependent)
- Data storage systems
- Observability and monitoring tools
- Geospatial data providers (e.g. environmental datasets)

## Selection Criteria

All subprocessors are selected based on:

- Security posture
- Availability guarantees
- Data handling transparency
- Enterprise reliability

## Transparency Principle

The system is designed to allow full visibility of:
- Data flow paths
- External dependencies
- Integration points

## Deployment Flexibility

Subprocessor configuration is not fixed and may vary depending on:
- Client deployment environment
- Regulatory requirements
- Infrastructure strategy
